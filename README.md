# 汉诺塔 极简至美版 (Tower of Hanoi - Premium Edition)

这是一个使用 Python 和 Tkinter 开发的高颜值汉诺塔交互游戏。

## ✨ 特性
- **现代美学设计**：深色模式背景，配合 Material Design 的缤纷色环圆盘。
- **顺滑动画**：支持圆盘移动动画，让操作逻辑更直观。
- **可交互操作**：点击柱子即可选取或放置圆盘。
- **自定义难度**：支持 3-8 个圆盘的难度调节。
- **自动演示**：内置递归算法，一键演示最优解法。
- **步数跟踪**：实时显示当前步数，并与完美步数对照。

## 🚀 如何运行
1. 确保已安装 Python 环境。
2. 在终端运行以下命令：
   ```bash
   python hanoi_game.py
   ```

## 🎮 操作指南
- **移动圆盘**：先点击源柱子顶部的圆盘，再点击目标柱子即可放置。注意大盘不能放在小盘上。
- **自动解题**：点击“自动演示”按钮，观看 AI 如何以最少步数完成游戏。
- **调节难度**：从下拉菜单中选择圆盘数量，点击“重新开始”以应用。
